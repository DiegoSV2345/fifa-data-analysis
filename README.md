# FIFA 23 – Análisis Exploratorio de Datos (Python)

Este proyecto presenta un análisis exploratorio de datos (EDA) sobre equipos de fútbol masculino utilizando el dataset FIFA 23, con el objetivo de identificar patrones de rendimiento, diferencias entre ligas y relaciones entre variables deportivas y económicas.

## Objetivo del proyecto
Analizar el rendimiento general de los equipos de fútbol y comprender cómo factores como la liga, el presupuesto de fichajes y la edad promedio influyen en el desempeño.

## Dataset
- Fuente: Kaggle – FIFA 23 Complete Player Dataset
- Archivo utilizado: male_teams.csv
- Registros: +385.000
- Variables: 54

## Metodología
1. Carga y exploración inicial de los datos
2. Limpieza y selección de variables relevantes
3. Análisis exploratorio y visualización
4. Interpretación de resultados y conclusiones

## Principales hallazgos
- El rendimiento de los equipos se concentra en rangos medios, con pocos equipos élite.
- Existen diferencias claras de nivel competitivo entre ligas.
- Se observa una relación positiva entre presupuesto de fichajes y rendimiento general.
- Los equipos con edades promedio intermedias presentan mejor desempeño.

## Tecnologías utilizadas

- pandas
