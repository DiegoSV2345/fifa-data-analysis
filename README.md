# FIFA 23 – Análisis Exploratorio de Datos (Python)

### Dataset utilizado
Para este análisis se utiliza el archivo `male_teams.csv`, el cual contiene información agregada de equipos de fútbol masculinos, permitiendo realizar un análisis exploratorio de datos de forma clara y didáctica.

## Objetivo del proyecto
- Cargar y explorar el dataset
- Comprender la estructura y calidad de los datos
- Identificar variables relevantes para el análisis
- Preparar los datos para etapas posteriores de limpieza y visualización

## Contenido del proyecto
- Carga y exploración inicial de datos
- Revisión de tipos de variables y valores nulos
- Análisis descriptivo de variables numéricas
- Preparación para la limpieza de datos

## Tecnologías utilizadas
- Python
- pandas
- matplotlib
- seaborn

## Dataset
Fuente: Kaggle – FIFA 23 Complete Player Dataset
